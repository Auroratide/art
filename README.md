# Art by Auroratide

I "draw" sometimes, \[tiny gallery of art is on my website](https://auroratide.com/art/).

"Draw", in quotes because I don't actually know how to draw properly. Instead, I manipulate hundreds of mathematical objects called Bézier Curves until they resemble something people might think of as "art". And the end result is a file called a Scalable Vector Graphic (SVG) that I can make as big or small as I want without blurring or resolution loss.

That might sound fancy, but it's really just an unhumble way of saying, "I move dots and shapes around on a screen."

## Tools

I primarily use \[Inkscape](https://inkscape.org/) nowadays, though I use to use vector layers in \[Krita](https://krita.org/en/). Everything you see saved here is from when I started using Inkscape.
